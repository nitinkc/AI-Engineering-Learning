# Weekly Revision System

## Why This Works

Interview performance improves when learning loops are short, repeated, and measured.

## Retention Cadence (1-3-7-14)

- Day 1: Learn and run one micro-task.
- Day 3: Recall from memory.
- Day 7: Explain to an imaginary interviewer.
- Day 14: Deliver full answer with tradeoffs and metrics.

## Weekly Scorecard

| Category | Target |
|---|---|
| Technical drills completed | >= 3 |
| STAR stories updated | >= 2 |
| Mock interview sessions | >= 1 |
| Weak answers fixed | >= 3 |

## End-of-Week Review

- What did I explain clearly?
- Where did I hesitate?
- Which metrics were missing?
- Which tradeoffs were weak?

## Micro-Lab

<details>
<summary>15-minute revision card</summary>

- Pick one topic (RAG, tools, evals, or deployment).
- Write 5 bullets from memory.
- Convert them into a 60-second answer.
- Compare to your reference notes and fill gaps.

</details>
